Features
Elixir WebRTC aims to provide you with all the pieces needed to create fully functional real-time audio/video applications.
Observability
Dedicated, easy to integrate with Phoenix LiveDashboard, server-side webrtc-internals.
Simulcast
Support for multiple inbound resolutions.
Xav
Library for audio/video decoding making it easy to feed data into Elixir Nx.

A series of tutorials introducing you to both WebRTC and Elixir WebRTC.
